HTML Terms

HTML

It defines the meaning and structure of web content

HTML elements

The html element is the root element and it defines the whole HTML document

HTML tag

the keywords on a web page that define how your web browser must format and display your web page.

Angle Bracket

commonly used to enclose a code of some type

HTML attribute

Attributes provide additional information about elements

HTML commant

The comment tag is used to insert comments in the source code.

HTML object elements

The tag defines a container for an external resource

HTML special charecter

specific pieces of HTML code designed to display characters that are used in the HTML code or to include characters that are not found on the keyboard in the text the viewer sees
